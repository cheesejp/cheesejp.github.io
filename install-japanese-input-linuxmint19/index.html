Linux Mintに日本語入力を追加する手順

2018-12-06

導入したOSのバージョン

Linux Mint 19 Tara

設定ウィンドウを開く

日本語をインストールする

インストールパッケージ一覧が表示されるのでインストールする。

パスワードの入力が求められるので入力する。

入力方法の変更

入力方法でIBusが選択できるようになるので選択する。

Linux Mintを再起動する

タスクバーからMozcを選択
